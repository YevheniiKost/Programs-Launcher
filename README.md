# Programs-Launcher

Невеликий Windows-лаунчер для робочих профілів:
- Створює профілі з наборів програм (Chrome, Slack, Unity, тощо).
- Запускає всі програми профілю однією кнопкою.
- Зберігає конфіг у `%AppData%\WorkLauncher\profiles.json`.
- Вміє додавати себе в автозапуск.

## Features

- Multiple profiles
- Editable list of applications per profile
- JSON config

## Installation

1. Встанови .NET Desktop Runtime (версія 8.0).
2. Скачай останній реліз з GitHub Releases.
3. Запусти `WorkLauncher.exe`.

## Usage

1. Вибери профіль.
2. Натисни **Run All**.
3. Через **Edit Profiles…** налаштуй потрібні програми.

## Config

Файл з конфігом зберігається в:
`%AppData%\WorkLauncher\profiles.json`
